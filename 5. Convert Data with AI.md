# 🤖 AI Converter

## 📌 Question

The DevOps AI Development Team is exploring how artificial intelligence can improve productivity and communication by transforming lengthy text into concise bullet points.

Create a Python-based AI module that converts a paragraph into short, meaningful bullet points using the OpenAI API.

### ✅ Requirements

- Create the file: `/root/openaiproject/converter.py`
- Initialize the OpenAI client using:
  - `OPENAI_API_KEY`
  - `OPENAI_API_BASE`
- Create a function:

```python
convert_to_bullets(text: str) -> str
```

- Use the following API parameters:
  - `model="openai/gpt-4.1"`
  - `max_tokens=150`
  - `temperature=0.1`
- Store the API response in a variable named `response`
- Print the bullet points to the console

### 📝 Paragraph to Convert

```text
Artificial Intelligence is transforming industries by automating tasks, improving decision-making, and enabling new innovations across healthcare, finance, and education.
```

## 💻 converter.py

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("OPENAI_API_KEY"),
    base_url=os.environ.get("OPENAI_API_BASE")
)

def convert_to_bullets(text: str) -> str:

    prompt = f"""
Convert the following paragraph into short, clear, and meaningful bullet points.

Paragraph:
{text}

Rules:
- Use concise language
- Keep bullet points short
- Do not add extra information
- Return only bullet points
"""

    response = client.chat.completions.create(
        model="openai/gpt-4.1",
        messages=[
            {
                "role": "user",
                "content": prompt
            }
        ],
        max_tokens=150,
        temperature=0.1
    )

    return response.choices[0].message.content.strip()


paragraph = """
Artificial Intelligence is transforming industries by automating tasks, improving decision-making, and enabling new innovations across healthcare, finance, and education.
"""

result = convert_to_bullets(paragraph)

print(result)
```

# 📖 Explanation

This project demonstrates how to use the OpenAI API with Python to convert long paragraphs into concise and readable bullet points.

The script:

1. Creates an OpenAI client using the API key and base URL from environment variables.
2. Defines a reusable function `convert_to_bullets(text)` that accepts paragraph text as input.
3. Builds a parameterized AI prompt dynamically using the provided text.
4. Sends the prompt to the OpenAI GPT-4.1 model.
5. Receives the AI-generated response containing bullet points.
6. Prints the formatted bullet points to the console.

This approach helps improve readability and productivity by summarizing long content into quick, easy-to-understand points.
