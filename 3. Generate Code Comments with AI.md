# 🤖 AI Comment Generator

Automatically generate meaningful one-line comments or docstrings for Python code snippets using the OpenAI API.

---

# 📌 Problem Statement

The DevOps AI Development Team is exploring how artificial intelligence can improve code readability and maintainability by automatically generating meaningful comments.

Build a Python-based AI module that:

- Accepts a code snippet
- Sends it to an OpenAI model
- Generates a concise one-line comment/docstring explaining the code

---

# ⚙️ Requirements

Inside `/root/openaiproject/commenter.py`:

- Create an OpenAI client using:
  - `API_KEY`
  - `BASE_URL`
- Define a function:

```python
generate_comment(code_snippet: str) -> str
```

The function should:

- Construct a parameterized AI prompt
- Send the prompt to:
  - Model: `openai/gpt-4.1-mini`
- Use:
  - `max_tokens=30`
  - `temperature=0.2`
- Store the API result in:

```python
response
```

- Print the generated comment
- Return the generated comment

# 💻 Python Script

```python
import os
from openai import OpenAI

# Create OpenAI client
client = OpenAI(
    api_key=os.environ.get("API_KEY"),
    base_url=os.environ.get("BASE_URL")
)

def generate_comment(code_snippet: str) -> str:
    """
    Generate a one-line comment for the provided code snippet.
    """

    prompt = f"""
Generate a clear and concise one-line Python comment or docstring
that explains the purpose of the following code:

{code_snippet}

Return only the comment.
"""

    response = client.chat.completions.create(
        model="openai/gpt-4.1-mini",
        messages=[
            {
                "role": "user",
                "content": prompt
            }
        ],
        max_tokens=30,
        temperature=0.2
    )

    generated_comment = response.choices[0].message.content.strip()

    print(generated_comment)

    return generated_comment


# Test code snippet
code = '''
def calculate_area(length, width):
    return length * width
'''

# Generate AI comment
generate_comment(code)
```
