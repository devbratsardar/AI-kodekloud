# 📧 AI Email Assistant

## 📌 Question

The Datacenter AI Development Team is working on automating written communication tasks.

As part of this initiative, build a Python-based AI Email Assistant that rewrites messages into polite and professional emails using the OpenAI API.

### ✅ Requirements

- Create the file: `/root/openaiproject/email_assistant.py`
- Initialize the OpenAI client using:
  - `OPENAI_API_KEY`
  - `OPENAI_API_BASE`
- Define a function:

```python
rewrite_email(text: str) -> str
```

- Use the following API parameters:
  - `model="openai/gpt-4.1-mini"`
  - `max_tokens=60`
  - `temperature=0.1`
- Store the API response in a variable named `response`
- Print the rewritten email to the console

### 📝 Email Text

```text
hey send me that report asap
```

python3 -m venv venv

## 💻 email_assistant.py

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("OPENAI_API_KEY"),
    base_url=os.environ.get("OPENAI_API_BASE")
)

def rewrite_email(text: str) -> str:
    prompt = f"""
Rewrite the following email in a polite and professional tone:

{text}
"""

    response = client.chat.completions.create(
        model="openai/gpt-4.1-mini",
        messages=[
            {"role": "user", "content": prompt}
        ],
        max_tokens=60,
        temperature=0.1
    )

    return response.choices[0].message.content


result = rewrite_email("hey send me that report asap")
print(result)
```

# 📖 Explanation

This project demonstrates how to use the OpenAI API to transform informal email messages into professional and polite communication.

The script:

1. Creates an OpenAI client using API credentials stored in environment variables.
2. Defines a reusable function `rewrite_email()` that accepts email text as input.
3. Generates a parameterized prompt asking the AI to rewrite the email professionally.
4. Sends the prompt to the GPT-4.1 Mini model.
5. Receives the rewritten email response from the AI.
6. Prints the professional email to the console.

This automation can help improve workplace communication by ensuring emails are clear, polite, and professional.
