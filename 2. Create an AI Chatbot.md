# AI Travel ChatBot using OpenAI API

This project demonstrates a simple AI-powered travel chatbot built using the OpenAI Python SDK.

The chatbot acts as a friendly travel guide that greets users and asks where they would like to travel.

# 🧠 chatbot.py

```python
import os
from openai import OpenAI

# Create OpenAI client
client = OpenAI(
    api_key=os.environ.get("OPENAI_API_KEY"),
    base_url=os.environ.get("OPENAI_API_BASE_URL")
)

# Prompt
prompt = "You are a friendly travel guide. Greet the user and ask where they want to go."

# Send request to model
nameresponse = client.chat.completions.create(
    model="openai/gpt-4.1-mini",
    messages=[
        {"role": "user", "content": prompt}
    ],
    temperature=0.7,
    max_tokens=100
)

# Print chatbot response
print(nameresponse.choices[0].message.content)
```
